Web Browsers

Google Chrome

Google Chrome is a freeware web browser that focuses on security, speed, and simplicity. It is developed by Google and utilizes the WebKit layout engine (Chrome v28+ will use Blink, a fork of WebKit, except on iOS) and the V8 JavaScript Rendering Engine.


    Google Chrome's Features:
  • Available for a variety of platforms; Windows, OS X, Linux, iOS, and Android
  • Comes prebundled with DevTools, a useful web developer tool that provides JavaScript debugging as well as many other capabilities.
  • Application performance and JavaScript speeds declared to be the fastest among web browsers at time of release.

Safari

Safari is an Apple Inc. web browser whose original purpose was to provide a defualt browser for Mac OS X. It utilizes Apple's own WebKit to render both layout and JavaScript. The two bundled technologies that come with WebKit that perform these tasks are WebCore and JavaScriptCore.


    Safari's Features:
  • Available for Windows, OS X, and iOS.
  • Comes prebundled with the Web Inspector, a DOM Inspector-like utility that developers browse the DOM of a web page.

Firefox

Firefox is a freeware web browser that focuses on current and anticipated web standards. It is developed by the Mozilla Foundation and utilizes the Gecko layout engine and the IonMonkey (OdinMonkey Firefox v22+) JavaScript Rendering Engine.


    Firefox's Features:
  • Available for a variety of platforms; Windows, OS X, Linux, and Android
  • Comes prebundled with the DOM Inspector, a useful web developer tool that allows web developers to detect HTML errors
  • As of February 11, 2011, Firefox has no security vulnerabilities.
