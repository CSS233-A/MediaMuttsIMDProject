Getting Started: Creating Your First Web Page

Create Your Home Page

  1. Open up the text editing program of your choice (NOT a word processing program such as Microsoft Office Word or LibreOffice), or if you do not have one, visit the Required Programs page and download the appropriate one for your operating system.
  2. Create a new file (if the text editor didn't automatically do it for you)
  3. Enter in the following information into the file you created: 
    <!DOCTYPE html>
<html>
	<head>
		<title> This is my first web page! </title>
	</head>

	<body>
		<h1>I am a header!</h1>
		
		<p>Hello world! I am some content.</p>
	</body>
</html>

  4. Save the file as index.html to the Desktop (Literally type index.html into the file name box and it will be saved properly)

    * NOTE: index.html is a special type of html file. A browser will attempt to find the index.html file if no other file is specified, i.e. it is the default name for an .html file.
  5. Minimize your text editor and locate the file that you saved to the Desktop. Once located, open the file in a browser (by default, just double-click the file and it will open in a browser)
  6. Congratulations! You have created your first web page and are ready to post it to your web space. Proceed to Getting Started: Uploading Your Web Page to finish with the Getting Started Tutorials.
