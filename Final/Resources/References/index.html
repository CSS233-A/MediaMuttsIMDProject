Useful Web References

W3Schools

W3Schools is a web developer information website, with tutorials and references relating to web development topics such as HTML, CSS, JavaScript, PHP, and SQL. It provides decent tutorials for beginners and can be a good starting point for learning HTML, CSS, and JavaScript.


    W3School's Features:
  • Tutorials for 43 different topics.
  • Refences for 25 different topics
  • Numerous examples for HTML, CSS, JavaScript, and more.

Mozilla Developer Network

Mozilla Developer Network (MDN) is the official Mozilla Foundation website for development documentation of web standards and Mozilla projects. MDN provides very in-depth references geared toward more knowledgable and experienced web developers. Extensive refences not found on w3schools can also be found here.


    MDN's Features:
  • Provides detailed references on for nearly all client-side markups and programming languages.
  • All references provided are all in-depth and highly specific.
  • Provides exquisite documentation on all scripting/programming methods.

Stack Overflow

Stack Overflow is a web forum that focuses on answering questions related to computer programming. The website serves as a platform for users to ask and answer questions, and, through membership and active participation, to vote questions and answers up or down and edit questions and answers in a fashion similar to a wiki. It is a very useful resource for the more abstract programming questions.


    Stack Overflow's Features:
  • Your question might have already been asked and answered by others and you have the ability to view it.
  • Ask a question and get it answered by a community driven to help other programmers.
