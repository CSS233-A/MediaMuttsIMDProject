Project Management

Google Sites

Google sites is a structured wiki- and web page-creation tool offered by Google as part of the Google Apps Productivity suite. The goal of Google Sites is for anyone to be able to create a team-oriented site where multiple people can collaborate and share files.


    Google Site's Features:
  • Custom Domain Name Mapping
  • Multi-Tier Permissions and Accessibility

GitHub

GitHub is a web-based hosting service for software development projects that use the Git revision control system. GitHub offers both paid plans for private repositories, and free accounts for open source projects.


    GitHub's Features:
  • Integrated Issue Tracking
  • Collaboration Between Large Teams
  • Code Review

SmartSheet

Smartsheet is an online project management, productivity and team collaboration software as a service application. Smartsheet combines functions found in Microsoft Excel, Project, Access, and Sharepoint into a single application.


    Smartsheet's Features:
  • Recording and organization of data
  • Unlimited, free collaborators
  • Task Management
  • Notifications, alerts, and emails
