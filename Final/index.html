Relevant Information

HTML5

Hypertext Markup Language (HTML) is a markup language for structuring and presenting content for the Web and a core technology of the internet.

    HTML gives authors the means to:
  • Publish online documents with headings, text, tables, lists, photos, etc.
  • Retrieve online information via hypertext links, at the click of a button.
  • Design forms for conducting transactions with remote services, for use in searching for information, making reservations, ordering products, etc.
  • Include spread-sheets, video clips, sound clips, and other applications directly in their documents.

CSS 3

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation semantics of a document written in a markup language.

    CSS gives the author the means to:
  • The affect the presentation of a web page by changing colors, layout, and fonts.
  • Adapt the web page to the screen size of the device viewing the page.
  • Change the layout of multiple pages of a site through independent "stylesheets"
